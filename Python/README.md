# AI Workforce Analytics using Python

## Project Overview
This project performs Exploratory Data Analysis (EDA) on an AI job market dataset using Python. The objective is to identify trends in salaries, AI adoption, automation risk, required skills, and company size.

## Tools & Libraries
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Analysis Performed
- Data Loading
- Data Cleaning
- Missing Value Check
- Duplicate Check
- Salary Distribution Analysis
- AI Adoption Level Analysis
- Automation Risk Analysis
- Top AI Skills Analysis
- Salary Distribution by Company Size
- Correlation Matrix

## Key Business Insights
- Most AI jobs fall within a competitive salary range.
- Medium AI adoption is the most common level.
- Automation risk differs across job roles.
- Python, Machine Learning, Data Analysis, and Project Management are among the most demanded skills.
- Salary trends vary across different company sizes.

## Conclusion
The analysis provides valuable insights into AI workforce trends and supports data-driven decision-making through visualizations and business insights.
