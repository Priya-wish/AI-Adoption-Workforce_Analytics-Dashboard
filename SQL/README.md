# SQL Analysis – AI Adoption & Workforce Analytics

## Overview

This folder contains SQL scripts used to explore, clean, and analyze the AI Adoption & Workforce Analytics dataset. The queries support business insights used in the Power BI dashboards and Python exploratory data analysis.

---

## Files Included

| File | Description |
|------|-------------|
| Database_setup.sql | Creates the database schema and imports the dataset. |
| Data_Exploration.sql | Performs initial data exploration, null value checks, and summary statistics. |
| Business_Analysis.sql | Contains business-focused SQL queries for salary, industry, location, and AI adoption analysis. |
| Advanced_SQL.sql | Demonstrates advanced SQL concepts including CTEs, Window Functions, Aggregations, Ranking, and Analytical Queries. |

---

## SQL Concepts Used

- SELECT Statements
- WHERE, ORDER BY
- GROUP BY & HAVING
- Aggregate Functions

- JOINS

- Window Functions

- Subqueries
