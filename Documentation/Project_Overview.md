AI Adoption & Workforce Analytics Dashboard

Project Overview

The objective of this project is to analyze the impact of Artificial Intelligence on workforce dynamics across industries and locations.

The project explores:

- AI Adoption Levels
- Salary Trends
- Workforce Distribution
- Automation Risk
- Remote Work Opportunities
- Future Job Growth

Using SQL, Python, and Power BI, the project aims to transform raw workforce data into actionable business insights.

Technologies Used

- MySQL
- Python
- Power BI
- Microsoft Excel
- GitHub

Key Objectives

- Identify industries leading AI adoption.
- Analyze salary variations across job roles.
- Evaluate automation risk by industry.
- Study remote work trends.
- Examine future workforce growth opportunities.
- Develop an interactive Power BI dashboard for decision-making.

Expected Outcomes

- Workforce trend analysis
- AI adoption insights
- Salary benchmarking
- Industry comparison
- Future job market projections
