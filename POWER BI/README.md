Power BI Dashboard

Overview

This folder contains the Power BI dashboard developed for the AI Adoption and Workforce Analytics Dashboard project.

The dashboard provides interactive visualizations to analyze AI workforce trends, salary distribution, industry demand, AI adoption, automation risk, remote work opportunities, and future job growth.

File Included

AI_Adoption_Workforce_Analytics.pbix

This Power BI file contains five interactive dashboards:

- Dashboard 1 – AI Workforce Analytics Overview
- Dashboard 2 – Salary Analysis
- Dashboard 3 – Skills & Industry Analysis
- Dashboard 4 – Location Analysis
- Dashboard 5 – AI Adoption & Future Workforce Trends

Features

- Interactive slicers and filters
- Cross-filtering between visuals
- KPI Cards
- Maps and geographic analysis
- Salary analysis
- AI adoption analysis
- Automation risk analysis
- Growth projection analysis

Software Required

- Microsoft Power BI Desktop (latest version recommended)

Notes

The dashboard was created using the dataset available in the Dataset folder of this repository and forms the visualization component of the complete analytics workflow.
